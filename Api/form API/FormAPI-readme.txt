
FormAPI 1.0 API doc

                          by johnbanq

本API是为在高级计算机/显示器上显示一些能工作的小控件/图形而设计
本API大多数的返回是通过event系统,小部分的控制也是通过event系统
API列表:

============ 控件类 ===============================
按钮[button]系:

form.ClickButton(按钮ID,输出按钮的位置[X值],输出按钮的位置[Y值],按钮长度,按钮上的字,按钮颜色,字体颜色,按下后的按钮颜色)
作用:在电脑屏幕上输出一个按钮
操作/使用方法：

按钮被按下后,会触发[Button_click]event,返回值: 1:按钮ID[数字]
关闭按钮:发送[Button_close],所携带的值: 1: 按钮ID[数字]

form.TouchButton(按钮ID,输出的屏幕方向,输出按钮的启始位置[X值],输出按钮的启始位置[Y值],按钮长度,按钮上的字,按钮颜色,字体颜色,按下后的按钮颜色)
作用:在给定的外接屏幕上输出一个按钮
操作/使用方法：

按钮被按下后,会触发[Button_Touch]event,返回值: 1:按钮ID[数字]
关闭按钮:发送[Button_close]event,所携带的值: 1: 按钮ID[数字]
修改按钮颜色: 发送[Button_Cchange]event,所携带的值: 1:按钮ID[数字] 2:按钮颜色[数字] 3:被按下后的按钮颜色[数字] 4:字体颜色[数字]

文本框[textbox]：

form.TextBox(文本框ID,输出文本框的启始位置[X值],输出文本框的启始位置[Y值],文本框长度,文本框颜色,文本框中字体颜色)

作用:在电脑屏幕上输出一个文本框,用户点击时会等待用户输入
操作/使用方法：

文本框被按下并被输入后,会触发[Textbox_input]event,返回值: 1:文本框ID[数字] 2:输入的东西[字符串]
关闭文本框:发送[Textbox_close]event,所携带的值: 1: 文本框ID[数字]

进度条[ProgressBar] 类:

form.ProgressBar(进度条ID,输出进度条的启始位置[X值],输出进度条的启始位置[Y值],进度条长度,进度条背景颜色,用来表示进度条被填的颜色)

作用:在电脑屏幕上输出一个进度条,会根据发送的值改变自己所表示的进度
操作/使用方法：

若要给进度条发送值:发送[ProgressBar_set]event,所携带的值: 1:进度条ID[数字] 2:进度条的值(填满部分的长度)[数字]
关闭进度条:发送[ProgressBar_close]event,所携带的值: 1: 进度条ID[数字]

form.TouchProgressBar(进度条ID,输出的屏幕方向,输出进度条的启始位置[X值],输出进度条的启始位置[Y值],进度条长度,进度条背景颜色,用来表示进度条被填的颜色)

作用:在电脑屏幕上输出一个进度条,会根据发送的值改变自己所表示的进度
操作/使用方法：

若要给进度条发送值:发送[ProgressBar_set]event,所携带的值: 1:进度条ID[数字] 2:进度条的值(填满部分的长度)[数字]
关闭进度条:发送[ProgressBar_close]event,所携带的值: 1: 进度条ID[数字]

============ 绘图类 ===============================

点[dot]类:

form.dot(输出点的位置[X值],输出点的位置[Y值],点的颜色)

作用:在电脑屏幕上输出1个点

form.Touchdot(输出的屏幕方向,输出点的位置[X值],输出点的位置[Y值],点的颜色)

作用:在给定的外接屏幕上输出1个点

线[line]类:

form.line(输出线的启始位置[X值],输出线的启始位置[Y值],线的长度,线的颜色)

作用:在电脑屏幕上输出1条线

form.Touchline(输出的屏幕方向,输出线的启始位置[X值],输出线的启始位置[Y值],线的长度,线的颜色)

作用:在给定的外接屏幕上输出1条线

长方形[block]类:

form.block(输出长方形的左上角位置[X值],输出长方形的左上角位置[Y值],长方形长度,长方形宽度,长方形颜色)

作用:在电脑屏幕上输出一个长方形

form.Touchblock(输出的屏幕方向,输出长方形的左上角位置[X值],输出长方形的左上角位置[Y值],长方形长度,长方形宽度,长方形颜色)

作用:在给定的外接屏幕上输出一个长方形

[tips:似乎这些都可以在程序注释中找到]
